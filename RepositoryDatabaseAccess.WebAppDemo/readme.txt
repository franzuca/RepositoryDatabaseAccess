RepositoryDatabaseAccess Demo

Setup:
1) Create a database on a server of your own
2) Complete DemoDbContext connection string data on web.config
3) Run "Enable-Migrations" on Package Manager Console
4) Run "Update-Database" to apply Code First Migrations for creating Company, Currency and Client tables

More info about code first setup: https://msdn.microsoft.com/en-us/library/jj193542(v=vs.113).aspx
